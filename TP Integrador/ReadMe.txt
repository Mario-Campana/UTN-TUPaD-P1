# Proyecto: Estructura Jerárquica y Algoritmos de Árbol Binario en Python

## Descripción

Este proyecto consiste en la implementación y manejo de una estructura de datos jerárquica basada en árboles binarios, desarrollada en Python. Se simula una jerarquía educativa conformada por una provincia, distritos, escuelas y aulas. Además, se incorporan funciones para recorrer el árbol (in-orden, pre-orden, post-orden), buscar elementos en la jerarquía y localizar alumnos mediante algoritmos de búsqueda lineal y binaria.

El trabajo se enmarca en un contexto educativo, buscando profundizar en el entendimiento de estructuras de datos recursivas y su aplicación práctica en programación.

Reflexiones del Equipo

Durante el desarrollo del proyecto, se reforzaron conceptos esenciales sobre estructuras de datos jerárquicas y técnicas de búsqueda. La colaboración en pareja permitió dividir tareas eficientemente, combinando habilidades para diseño, programación y prueba de código.

Se enfrentaron desafíos en el manejo de recursividad y la visualización clara de la estructura, los cuales se resolvieron mediante pruebas y trabajo conjunto. Se aprendió que la implementación práctica de estos conceptos fortalece la comprensión teórica y prepara para abordar problemas complejos en programación.

Como posible mejora futura, se consideraría la creación de una interfaz gráfica para facilitar la interacción y una representación basada en clases para una estructura más flexible y escalable.
